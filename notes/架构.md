# 信息安全学习助手 Agent 项目方案

## 1. 项目目标

本项目旨在实现一个面向信息安全学习场景的轻量级 Agent 系统。

系统支持：

- 安全概念解释
- 基于本地知识库的 RAG 问答
- 学习路线建议
- 自动问题路由

目标是在实践中理解：

- RAG
- Router
- Prompt Engineering
- LLM 调用
- Memory
- Agent Workflow

---

## 2. 系统模块

系统主要包括：

### 1. Router
负责判断用户问题类型。

支持分类：

- concept_explanation
- rag_qa
- study_advice
- web_search

---

### 2. Retriever
从本地知识库中检索相关文本块。

---

### 3. LLM
根据 Prompt 生成最终回答。

---

### 4. Advisor
生成信息安全学习路线建议。

---

### 5. Memory
保存最近几轮对话上下文。

---

### 6. Router Classifier
后期将 Router 从规则判断升级为可训练文本分类器。

---

## 3. 系统工作流程

用户输入问题

↓

Router 判断问题类型

↓

分发到对应模块

↓

生成回答

↓

保存到 Memory

---

## 4. 技术路线

项目计划采用：

- Python
- Prompt Engineering
- RAG
- TF-IDF
- SVM Router Classifier
- 本地知识库
- LLM API

---

## 5. 项目亮点

本项目不仅实现基础 RAG Agent，还设计了：

规则 Router
→ LLM Router
→ 自训练 Router 分类器

体现系统逐步升级过程。
